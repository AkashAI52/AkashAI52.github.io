GATE & Related Exam Resources

This section contains syllabus and curated resources for various exams such as GATE, TIFR, JEST, ISI, and CMI, aimed at students interested in theoretical computer science, machine learning, and mathematics.

GATE CS

Section 1: Engineering Mathematics

Syllabus:

Discrete Mathematics: Propositional and first order logic. Sets, relations, functions, partial orders and lattices. Monoids, Groups. Graphs: connectivity, matching, colouring. Combinatorics: counting, recurrence relations, generating functions.
Linear Algebra: Matrices, determinants, system of linear equations, eigenvalues and eigenvectors, LU decomposition.
Calculus: Limits, continuity and differentiability, Maxima and minima, Mean value theorem, Integration.
Probability: Random variables, Uniform, normal, exponential, Poisson and binomial distributions. Mean, median, mode and standard deviation. Conditional probability and Bayes theorem.

Books:

  • Discrete Mathematics and Its Applications – Rosen
  • Elements of Discrete Mathematics – C. L. Liu
  • Higher Engineering Mathematics – H.K.DASS
  • Advanced Engineering Mathematics – ERWIN KREYSZIG

Lecture Videos:

  • Discrete Mathematics, IIIT Bangalore Prof. Ashish Choudhury
  • Engineering Mathematics - I, IIT Kharagpur Prof. Jitendra Kumar

Section 2: Digital Logic

Syllabus:

Combinational and sequential circuits. Minimization. Number representations and computer arithmetic (fixed and floating point).

Books:

  • Digital Design and Computer Design – Morris Mano

Lecture Videos:

  • Digital Circuits & Systems by Prof.S.Srinivasan

Section 3: Computer Organization and Architecture

Syllabus:

Machine instructions and addressing modes. ALU, data‐path and control unit. Instruction pipelining, pipeline hazards. Memory hierarchy: cache, main memory and secondary storage; I/O interface (interrupt and DMA mode).

Books:

  • Computer Organization – Carl Hamacher

Lecture Videos:

  • Computer Architecture, IIT Delhi Prof. Smruti Ranjan Sarangi
  • COA 2.0 - Vishvadeep Gothi

Section 4: Programming and Data Structures

Syllabus:

Programming in C. Recursion. Arrays, stacks, queues, linked lists, trees, binary search trees, binary heaps, graphs.

Books:

  • Data Structures – Cormen / CLRS

Section 5: Algorithms

Syllabus:

Searching, sorting, hashing. Asymptotic worst case time and space complexity. Algorithm design techniques: greedy, dynamic programming and divide‐and‐conquer. Graph traversals, minimum spanning trees, shortest paths.

Books:

  • Introduction to Algorithms – CLRS

Lecture Videos:

  • Add link

Section 6: Theory of Computation

Syllabus:

Regular expressions and finite automata. Context-free grammars and push-down automata. Regular and context-free languages, pumping lemma. Turing machines and undecidability.

Books:

  • An Introduction to FORMAL LANGUAGES and AUTOMATA - PETER LINZ
  • Introduction to the Theory of Computation – Michael Sipser
  • Introduction to Automata Theory, Languages, and Computation – Hopcroft, Motwani, Ullman

Lecture Videos:

  • Theory of Computation, IIT Kanpur Prof. Raghunath Tewari
  • Standard Video Lectures - Ullman

Section 7: Compiler Design

Syllabus:

Lexical analysis, parsing, syntax-directed translation. Runtime environments. Intermediate code generation. Local optimization, Data flow analyses: constant propagation, liveness analysis, common sub expression elimination.

Books:

  • Principles Of Compiler Design – Aho (Dragon Book)

Lecture Videos:

  • Compiler Design by Sorav Bansal - IIT Delhi(~ 100 videos)

Section 8: Operating Systems

Syllabus:

System calls, processes, threads, inter‐process communication, concurrency and synchronization. Deadlock. CPU and I/O scheduling. Memory management and virtual memory. File systems.

Books:

  • Operating System Concepts – Galvin

Lecture Videos:

  • Operating Systems - Smruti R. Sarangi

Section 9: Databases

Syllabus:

ER‐model. Relational model: relational algebra, tuple calculus, SQL. Integrity constraints, normal forms. File organization, indexing (e.g., B and B+ trees). Transactions and concurrency control.

Books:

  • Database System Concepts – Korth

Lecture Videos:

  • Introduction to Database Systems, IIT Madras Prof. Sreenivasa Kumar

Section 10: Computer Networks

Syllabus:

Concept of layering: OSI and TCP/IP Protocol Stacks; Basics of packet, circuit and virtual circuitswitching; Data link layer: framing, error detection, Medium Access Control, Ethernet bridging; Routing protocols: shortest path, flooding, distance vector and link state routing; Fragmentation and IP addressing, IPv4, CIDR notation, Basics of IP support protocols (ARP, DHCP, ICMP), Network Address Translation (NAT); Transport layer: flow control and congestion control, UDP, TCP, sockets; Application layer protocols: DNS, SMTP, HTTP, FTP, Email.

Books:

  • Computer Networking: A Top-Down Approach – Kurose & Ross

Lecture Videos:

  • Computer Networks - GATE Computer Science - Ravula

GATE AI / Data Science Syllabus

Section 1: Probability and Statistics

Syllabus:

Counting (permutation and combinations), probability axioms, Sample space, events, independent events, mutually exclusive events, marginal, conditional and joint probability, Bayes Theorem, conditional expectation and variance, mean, median, mode and standard deviation, correlation, and covariance, random variables, discrete random variables and probability mass functions, uniform, Bernoulli, binomial distribution, Continuous random variables and probability distribution function, uniform, exponential, Poisson, normal, standard normal, t-distribution, chi-squared distributions, cumulative distribution function, Conditional PDF, Central limit theorem, confidence interval, z-test, t-test, chi-squared test.

Books:

  • Probability – Sheldon Ross
  • All of Statistics – Larry Wasserman

  • Add link

Section 2: Linear Algebra

Syllabus:

Vector space, subspaces, linear dependence and independence of vectors, matrices, projection matrix, orthogonal matrix, idempotent matrix, partition matrix and their properties, quadratic forms, systems of linear equations and solutions; Gaussian elimination, eigenvalues and eigenvectors, determinant, rank, nullity, projections, LU decomposition, singular value decomposition.

Books:

  • Linear Algebra – Gilbert Strang
  • Matrix Computations – Golub & Van Loan

Functions of a single variable, limit, continuity and differentiability, Taylor series, maxima and minima, optimization involving a single variable.

Section 4: Programming and Data Structures

Syllabus:

Programming in Python, basic data structures: stacks, queues, linked lists, trees, hash tables; Search algorithms: linear search and binary search, basic sorting algorithms: selection sort, bubble sort and insertion sort; divide and conquer: mergesort, quicksort; introduction to graph theory; basic graph algorithms: traversals and shortest path.

Section 5: Database Management and Warehousing

Syllabus:

ER-model, relational model: relational algebra, tuple calculus, SQL, integrity constraints, normal form, file organization, indexing, data types, data transformation such as normalization, discretization, sampling, compression; data warehouse modelling: schema for multidimensional data models, concept hierarchies, measures: categorization and computations.

  • Data Mining – Han & Kamber

Section 6: Machine Learning

Syllabus:

(i) Supervised Learning: regression and classification problems, simple linear regression, multiple linear regression, ridge regression, logistic regression, k-nearest neighbour, naive Bayes classifier, linear discriminant analysis, support vector machine, decision trees, bias-variance trade-off, cross-validation methods such as leave-one-out (LOO) cross-validation, k-folds cross-validation, multi-layer perceptron, feed-forward neural network;
(ii) Unsupervised Learning: clustering algorithms, k-means/k-medoid, hierarchical clustering, top-down, bottom-up: single-linkage, multiple-linkage, dimensionality reduction, principal component analysis.

Books:

  • Pattern Recognition and Machine Learning – Bishop
  • Hands-On ML – Aurélien Géron

CMI MSc / PhD Computer Science

Section 1: Discrete Mathematics

Syllabus:

Combinatorics, principle of induction, pigeonhole principle, permutations and combinations, finite set theory, functions, relations.

Section 2: Logic

Syllabus:

Boolean logic, truth tables, boolean circuits, AND, OR, NOT, NAND gates.

Section 3: Graph Theory

Syllabus:

Trees, bipartite graphs, matchings, BFS, DFS, minimum spanning trees, shortest paths.

Section 4: Formal Languages and Automata Theory

Syllabus:

Regular expressions, DFA, NFA, subset construction, pumping lemma, CFGs, computability basics.

Section 5: Algorithms

Syllabus:

Time complexity, recurrence relations, sorting (bubble, quick, merge, heap), searching.

Section 6: Algorithmic Techniques

Syllabus:

Dynamic programming, divide and conquer, greedy methods.

Section 7: Data Structures

Syllabus:

Lists, stacks, queues, binary search trees, heaps.

Section 8: Probability Theory

Finite probability spaces, conditional probability, Bayes theorem, expectation, variance, moments, Markov and Chebyshev inequalities.

Syllabus:

Limits, continuity, differentiability, multivariable calculus, partial derivatives, integration.

Section 10: Linear Algebra

Syllabus:

Vector spaces, linear operators, eigenvalues, eigenvectors, determinants, inner product spaces, symmetric and Hermitian matrices.

ISI MTech Computer Science

Section 1: Discrete Mathematics

Syllabus:

Logical reasoning, functions and relations, divisibility, congruences, primality, permutations and combinations, binomial and multinomial theorem, recurrence relations.

Section 2: Design and Analysis of Algorithms

Syllabus:

Asymptotic notation, searching, sorting, graph traversal, minimum spanning tree, shortest path algorithms.

Section 3: Data Structures

Syllabus:

Arrays, stacks, queues, linked lists, binary trees, heaps, AVL trees, B-trees.

Section 4: Graph Theory

Syllabus:

Graphs, adjacency matrix and list, subgraphs, paths and cycles, trees, connectivity, digraphs, vertex colouring, planar graphs.

Section 5: Formal Languages and Automata Theory

Syllabus:

Finite automata, regular expressions, pumping lemma, pushdown automata, context-free grammar, Turing machines, undecidability.

Section 6: Switching Theory and Logic Design

Syllabus:

Boolean minimization, Karnaugh maps, Quine-McCluskey, adders, multiplexers, ROM, flip-flops, counters, registers, FSM design.

Section 7: Computer Organization and Architecture

Syllabus:

Number systems, arithmetic, memory organization, I/O systems, processor design, pipelining.

Section 8: Operating Systems

Syllabus:

Memory management, process management, synchronization, deadlocks, file systems, device management.

Section 9: Database Management Systems

Syllabus:

Relational model, relational algebra, SQL, functional dependencies, normalization (2NF, 3NF, BCNF).

Section 10: Computer Networks

Syllabus:

OSI model, LAN topologies, MAC protocols, WAN technologies, encoding/decoding, flow control, error detection and correction.

Recommended Books

  • Operating Systems – Galvin
  • Computer Networks – Kurose & Ross
  • Database Systems – Korth
